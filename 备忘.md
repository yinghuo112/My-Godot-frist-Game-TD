## 自动地图生产

| 方案 | 实践次数 | 优点 | 缺点 | 选型 |
|------|---------|------|------|------|
| TileMapDual | 1 | 路径可控 / 瓦片资源少(47→15) / 与数据平衡无缝衔接 | 手动绘制，多图稍慢 | **当前推荐** |
| WFC 插件 | 0 | 批量生成快 / 风格统一 | 路径不可控 / 需人工验收 / 学成本高 | 备选 |

### 地图目录结构
```
res://
 ├── addons/
 │   ├── map_generator/
 │   │   └── MapGenerator.gd         路径生成算法（蛇形/随机漫步 + 2x2塔槽扫描）
 │   └── map_panel/
 │       └── plugin.gd               编辑器底部面板"地图控制"（扩展/清除/生成）
 ├── Scene/
 │   ├── tower_slot.tscn            2x2塔槽独立场景（半透明视觉 + Area2D点击 + 塔管理）
 │   ├── tower_slot.gd              TowerSlot 脚本（clicked/place_tower/remove_tower）
 │   ├── game_level.tscn            关卡主场景（Node2D + TileMapLayer，下一阶段）
 │   └── game_level.gd              关卡总指挥（下一阶段）
 ├── config/
 │   └── MapData.gd                 地图数据资源（map_seed / path_points / slot_positions）
 ├── 核心/
 │   ├── main.gd                    游戏主脚本（_place_tower / _on_slot_clicked）
 │   ├── map_manager.gd             地图管理器（load_map / get_slots / build_tower_at）
 │   └── slot_data.gd               槽位数据资源（position / difficulty）
 └── data/maps/
     ├── map_001.tres               初始地图数据（9 槽）
     └── map_002.tres               第 2 张图数据模板（8 槽，蛇形路径）
```

### 多地图基础设施（已完成）
- `config/MapData.gd` — 地图数据资源（map_seed / path_points / slot_positions / slot_difficulties）
- `核心/map_manager.gd` — `load_map()` / `get_slots()` / `build_tower_at()` / `free_slot_at()`
- `核心/slot_data.gd` — 单槽位数据资源（position / difficulty）
- `Scene/tower_slot.tscn` — 2x2 塔槽独立场景（半透明视觉 + Area2D 点击 + place_tower 管理塔子节点）
- `addons/map_generator/MapGenerator.gd` — 随机地图生成（蛇形/随机漫步 + 2x2 槽位扫描防重叠 + 动态 tileset 渲染）
- `addons/map_panel/plugin.gd` — 编辑器底部面板，支持扩展/清除草地、种子参数、生成新地图
- `data/maps/map_001.tres` — 初始地图数据（9 槽）
- `data/maps/map_002.tres` — 第 2 张图数据模板（8 槽，中央蛇形路径）

### 创建新地图
- 编辑器：在"地图控制"面板输入种子 → 点击"🎲 生成新地图"，自动创建 TileMap 瓦片 + EnemyPath 曲线 + TowerSlots 节点
- 运行时：`MapGenerator.generate(tilemap, md)` 返回 MapData，`map_manager.load_map(md)` 创建 TowerSlot 实例
